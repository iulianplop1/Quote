Quote
A Daily Movie Quotes App that distills cinematic wisdom into a daily routine.

Why I Built This
I'm a huge cinephile, but I found that the lines that really stuck with me were always buried deep inside massive scripts. I wanted a way to easily pull out those hidden gems so I can revisit them exactly when I need a spark of inspiration.

This app uses AI to do the heavy lifting, parsing through dense movie scripts to find quotes with high narrative impact. It automates the process so I can manage my entire script library and rediscover iconic lines from movies like The Dark Knight or Inception in a clean, minimal interface. It's about turning long scripts into a small, daily digital experience.

It was an excellent technical challenge to integrate AI script parsing and build a fluid morning routine player using React and Supabase.

Features
AI Script Analysis: Uses Gemini 2.5 Flash to automatically read scripts and extract the best quotes.
Morning Routine Integration: Create scheduled routines that play your favorite quotes alongside a background song to start your day.
Thematic Discovery: Search your script library by abstract themes like "justice," "fear," or "hope" to find relevant quotes instantly.
Cinematic Context: View the exact scene surrounding any quote to understand the character's motivation and the plot context.
Secure Backend: Built with Supabase for secure data storage and authentication.
